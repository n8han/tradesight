Scenario

You are given 5 time series of daily returns corresponding to 5 different stocks. With these 5 return series you can calculate the returns of a portfolio of any combination of these 5 stocks. For instance, if you were to hold the 5 stocks in equal amounts so that each stock was was 20% of your total portfolio, you could calculate the returns of this portfolio by summing the returns of each stock scaled by 20% at each point in time. This is an "equal weighted portfolio."

If you want to vizualize the performance of a time series of returns, one good way is to compute the cumulative return index. This is a series that shows the accumulation of each subsequent days' return.

Problems

  1. Compute the returns for the equally-weighted portfolio of these 5 stocks (a portfolio where each series contributes 20% to the total).
  2. Compute and graph the cumulative return index (by accumulating each day's returns from #1)
  3. Figure out:
    • the continuous time range over which the equal-weight portfolio in #2 increases by the max percentage
    • ... decreases by the minimum percentage?
  4. Apply your logic in #3 to figure out the 5 best "winning" trades - one in each stock - in which you "trade into" and "trade out" just once.
  5. Do the same for the worst "losing" trades
  6. Graph the "best trade", "worst trade", "equally weighted" return indexes. What does the graph look like?
